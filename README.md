# ghostway
A simple multi-layer redirector infrastructure automation.

## Overview
The project ***is currently under active development*** with the aim of creating a foundational, multi-layered redirector infrastructure. It's made to help beginners understand and handle the basics fo a simple yet sophisticated redirector setup.
The project focused on integration with the already published [ad-training-lab](https://github.com/brmkit/ad-training-lab), offering an hands-on and practical approach to mastering the essentials of red teaming... *while having fun.*
